Esta pagina trata de recetas de cocina con nombre DELICE. 
Dentro de esta pagina muestra 4 opciones de recetas como es: 
recetas de cocina, postres, bebidas y comida gourmet. Cada una de estas recetas direcciona a una pagina web que muestra la receta de la opcion elegida.

Dentro de esta carpeta se encuentran los archivos HTML que son el index y cada uno de las paginas donde se mujestran las opciones de recetas(bebidas, postres, gourmet, recetas de cocina, etc).
Se encuentra la carpeta de img, que dentro de esta se encuentran las imagenes y carpetas que contienen mas imagenes que se utilizaron para el desarrollo de la pagina web.
Se muestra tambien la carpeta css que son los archivos que se utilizaron para el diseño que compone cada una de las partes de la pagina.
Se encuentra tambien la pagina js donde se encuentran los archivos de javascript que se utilizaron para hacer la pagina web un poco mas dinamica, junto con bootstrap.
Para el desarrollo de la pagina se utilizo HTML, CCS, JQUERY, BOOTSTRAP, IMG, ETC.
